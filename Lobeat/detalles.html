¿Qué es LoBeat?

LoBeat es una aplicación web que permitirá a sus usuarios poder compartir su música de distintas maneras, para así poder conocer a nuevas personas que tengan gustos musicales similares. Ese intercambio de información se realiza con un enfoque social, es decir, LoBeat pretende afianzar con su uso nuevas relaciones de forma similar a aplicaciones de citas como Tinder, Meetic, Bumble, etc.

Para su correcto funcionamiento, los usuarios deben vincular su cuenta a una plataforma de streaming de música, que será el proveedor principal de información y permitirá conocer con exactitud la canciones, playlists, y en general gustos y preferencias de cada usuario. LoBeat se encarga por tanto de establecer un sistema que hace uso de esa información, y la explota de múltiples formas.
Por ejemplo, se introduce el concepto de “Love Playlist”, una lista de reproducción personalizada que cada usuario podrá utilizar como punto de partida para conectar con diferentes personas.

Usuarios y tareas:

  • Administradores: Usuarios con permisos totales que cuentan con acceso a todas las secciones de la aplicación y tienen permiso para modificar o eliminar cualquier información de la base de datos.

  • Usuarios registrados: Usuarios que ya están registrados en la página web y tienen acceso a los servicios que se ofertan. De forma complementaria, un usuario registrado puede seleccionar para su cuenta la modalidad para establecimientos o locales (Business User). Los usuarios con esta modalidad aparecerán como tal al resto de usuarios y pueden hacer uso de las mismas funcionalidades pero con un enfoque distinto. Podrán por ejemplo hacer pública la música que se escucha frecuentemente en el local, las canciones que se están escuchando en ese momento, etc.

  • Usuarios no registrados: Para poder tener acceso a todos los servicios, los usuarios tendrán que conectar su cuenta a una plataforma de streaming de música. Por tanto, los usuarios no registrados se encontrarán con una pantalla de login o registro.
